# Формирование торгового сценария

**ВАЖНО:** При формировании торгового сценария обязательно используй детальные правила из @trends.

## Задача

Создать детальный торговый сценарий для инструмента на основе уровней, предпосылок и анализа трендов (см. @trends).

## Обязательные элементы

### 1. Направление (см. @trends)

**Обязательно проверить перед выбором направления:**
- **Глобальный тренд (W1/MN):** определить общее направление рынка (см. @trends)
- **Локальный тренд (D1/H1):** определить краткосрочное направление (см. @trends)
- **Сила тренда:** оценить силу тренда (ценовое действие + объемы + ATR) (см. @trends)
- **Фаза рынка:** определить текущую фазу (накопление/импульс/коррекция/ретест/флэт/истощение) (см. @trends)

**Правило:** Предпочитать сделки в направлении глобального тренда (см. @trends).

- **Long**: Покупка (ожидание роста) — только если глобальный/локальный тренд восходящий
- **Short**: Продажа (ожидание падения) — только если глобальный/локальный тренд нисходящий

### 2. Триггер входа

Чёткое условие для входа:
- "Пробой 185.50 с ретестом"
- "Отскок от 180.00 с подтверждением объёма"
- "Пробой 185.50, затем возврат к 185.50 и отскок"

### 3. Точка входа

Конкретная цена или условие:
- "185.60 (после пробоя)"
- "180.10 (при отскоке от 180.00)"

### 4. Стоп-лосс (см. @trends)

**ВАЖНО:** Использовать правило приоритета технического стопа (см. @trends).

**Правила:**
- За уровнем поддержки/сопротивления
- Минимум 0.2% от цены входа
- Минимум 0.5 ATR от входа (избежать рыночного шума)
- Максимум 1% риска от депозита

**Расчет стоп-лосса:**

1. **Рассчитать технический стоп:**
   - Для лонга: чуть ниже уровня поддержки
   - Для шорта: чуть выше уровня сопротивления

2. **Рассчитать расчетный стоп:**
   - **20-30% от ATR(5)** (например, если ATR = $1.00, то расчетный стоп = $0.20 - $0.30)
   - Учитывать волатильность инструмента

3. **Сравнить и выбрать:**
   - Если **технический стоп ≤ расчетный стоп** → использовать **технический стоп**
   - Если **технический стоп > расчетный стоп** → пересмотреть точку входа или использовать расчетный стоп (но не превышать максимальный риск 1%)

**Пример:**
- Вход в лонг: $185.60
- Уровень поддержки: $184.90
- Технический стоп: $184.80 (за уровнем)
- ATR(5): $0.70
- Расчетный стоп: 20-30% от ATR = $0.14 - $0.21, значит стоп на уровне $185.60 - $0.20 = $185.40
- **Выбор:** Технический стоп $184.80 < расчетный $185.40 → использовать технический $184.80

### 5. Тейк-профит

Правила:
- Минимум 2:1 R:R (риск:прибыль)
- Желательно несколько уровней (TP1, TP2)
- Размещать у следующих уровней

### 6. Размер позиции

Рассчитать на основе риска:
```
Размер позиции = (Риск в $) / (Цена входа - Стоп-лосс)
```

Где:
- Риск в $ = Депозит × 1% (или меньше)
- Цена входа = Плановая цена входа
- Стоп-лосс = Цена стоп-лосса

## Процесс

1. Определить направление на основе (см. @trends):
   - **Глобального тренда (W1/MN)** — приоритет
   - **Локального тренда (D1/H1)**
   - **Силы тренда** (ценовое действие + объемы + ATR)
   - **Фазы рынка** (избегать торговли во флэте в середине диапазона)
   - Положения относительно уровней
   - Предпосылок

2. Определить триггер (см. @trends):
   - Пробой уровня (с подтверждением объемом — контрастный объем 2-3x среднего)
   - Отскок от уровня (с подтверждением объемом)
   - Пробой с ретестом
   - **Важно:** Триггер должен быть в направлении глобального тренда

3. Рассчитать точку входа:
   - После пробоя: цена пробоя + небольшой буфер
   - При отскоке: цена уровня + небольшой буфер

4. Разместить стоп-лосс (см. @trends):
   - Рассчитать технический стоп (за уровнем поддержки/сопротивления)
   - Рассчитать расчетный стоп (20-30% от ATR(5))
   - Выбрать меньший из двух (приоритет технического, если не превышает расчетный)
   - Проверить, что риск ≤ 1%

5. Разместить тейк-профит:
   - TP1: Минимум 2:1 R:R
   - TP2: 4:1 или 5:1 R:R
   - У следующих уровней

6. Рассчитать размер позиции:
   - На основе риска
   - Проверить, что риск ≤ 1%

## Формат вывода

```markdown
## Торговый сценарий

**Направление**: Long
**Триггер**: Пробой 185.50 с ретестом и подтверждением объёма
**Точка входа**: 185.60 (после пробоя)
**Стоп-лосс**: 184.90 (-0.38% от входа)
**Тейк-профит 1**: 187.00 (R:R 2:1)
**Тейк-профит 2**: 189.50 (R:R 5.5:1)
**Размер позиции**: 142 акции (риск: $100 = 1% депозита)

**Примечания**:
- Следить за новостями в 14:30 (CPI)
- Не входить до выхода новостей
- Если пробой слабый (низкий объём), пропустить
```

## Важные правила

1. **Проверять глобальный и локальный тренд** — предпочитать сделки в направлении глобального тренда (см. @trends)
2. **Оценивать силу тренда** — ценовое действие + объемы + ATR (см. @trends)
3. **Определять фазу рынка** — избегать торговли в середине диапазона (флэт) (см. @trends)
4. **Минимум 2:1 R:R** — иначе не торговать
5. **Максимум 1% риска** — на сделку
6. **Чёткий триггер** — должно быть ясно, когда входить
7. **Стоп-лосс обязателен** — для каждой сделки
8. **Приоритет технического стопа** — использовать технический стоп, если он не превышает расчетный (20-30% от ATR) (см. @trends)
9. **Подтверждение объемом** — движения без объема подозрительны (см. @trends)
