# Методология Герчика - Основные принципы

## Философия торговли

### Главные принципы

1. **Торговля — это бизнес**, а не азартная игра
2. **Дисциплина важнее таланта** — следование правилам критично
3. **Риск-менеджмент — основа** — без правильного управления рисками невозможно быть прибыльным
4. **Статистика — учитель** — анализ результатов помогает улучшать торговлю

## Обязательные правила

### 1. Домашнее задание

**Никогда не торговать без подготовки.**

Домашнее задание включает:
- Скрининг инструментов
- Определение уровней
- Проверку предпосылок
- Формирование торговых сценариев
- Составление списка инструментов на день

**Без домашнего задания — не торговать.**

### 2. Торговый план

**Все сценарии должны быть расписаны до начала сессии.**

План включает:
- Направление (Long/Short)
- Триггер входа
- Точку входа
- Стоп-лосс
- Тейк-профит
- Размер позиции

**План не меняется во время сделки** (кроме обоснованных случаев).

### 3. Риск-менеджмент

**Максимальный риск на сделку: 1% от депозита.**

Правила:
- Всегда использовать стоп-лосс
- Минимум 2:1 R:R (риск:прибыль)
- Максимум 3-5% риска в день
- После 3 убыточных сделок — прекратить торговлю

### 4. Эмоциональный контроль

**Эмоции — враг трейдера.**

Правила:
- Торговать только по плану
- Не увеличивать размер позиции после убытка
- Не удалять стоп-лосс
- Делать перерывы после убыточных сделок

## Подход к анализу

### Работа с уровнями

1. **Определять уровни заранее** — до начала сессии
2. **Использовать несколько типов уровней** — исторические, круглые, вчерашние
3. **Проверять силу уровня** — чем больше отскоков, тем сильнее
4. **Ждать подтверждения** — не входить сразу, ждать пробоя/отскока

### Выбор инструментов

Критерии:
- Достаточная волатильность (ATR)
- Достаточный объём (выше среднего)
- Чёткий тренд или граница диапазона
- Есть уровни для работы

### Точки входа

1. **У уровня** — вход при отскоке от поддержки/сопротивления
2. **После пробоя** — вход после пробоя уровня с подтверждением
3. **С подтверждением** — объём, разворотная свеча, другие сигналы

## Ведение статистики

**Статистика — основа улучшения.**

Обязательно вести:
- Журнал всех сделок
- Анализ результатов
- Выявление ошибок
- Корректировку стратегии

## Важные напоминания

1. **Торговля — это марафон, а не спринт** — важна долгосрочная прибыльность
2. **Не все дни будут прибыльными** — убыточные дни нормальны
3. **Дисциплина важнее таланта** — следование правилам критично
4. **Учиться на ошибках** — анализировать и корректировать

## Когда НЕ торговать

1. **Без домашнего задания** — не подготовился, не торгуй
2. **После 3 убыточных сделок** — прекратить на день
3. **При эмоциональном состоянии** — злость, страх, жадность
4. **Без чёткого плана** — если нет сценария, не торговать
5. **Перед важными новостями** — если не знаешь, как торговать новости
