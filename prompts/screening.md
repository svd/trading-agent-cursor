# Скрининг инструментов

## Задача

Найти инструменты, подходящие для торговли сегодня по методологии Герчика.

## Критерии отбора (см. @trends)

1. **Волатильность**: ATR(5) выше среднего за последние 20 дней (см. @trends)
2. **Объём**: Текущий объём выше среднего за последние 20 дней (минимум 1.2x, желателен контрастный объем 2-3x) (см. @trends)
3. **Глобальный тренд (W1/MN)**: Определить общее направление рынка на старших таймфреймах (см. @trends)
4. **Локальный тренд (D1/H1)**: Чёткий тренд на D1 или H4, либо инструмент у границы диапазона (см. @trends)
5. **Сила тренда**: Оценить силу тренда (ценовое действие + объемы + ATR) (см. @trends)
6. **Фаза рынка**: Определить текущую фазу (накопление/импульс/коррекция/ретест/флэт/истощение) — избегать инструментов во флэте в середине диапазона (см. @trends)
7. **Ликвидность**: Спред не более 0.1% для акций, 0.05% для фьючерсов

## Процесс

1. Получить список инструментов для скрининга (или использовать предопределённый список)
2. Для каждого инструмента:
   - Получить текущую котировку (`get_real_time_quote`)
   - Получить исторические данные (`get_stock_history` для анализа тренда на разных таймфреймах: W1, D1, H1)
   - Получить статистику (`get_market_stats` для ATR(5), объёмов)
   - Определить глобальный тренд (W1/MN) (см. @trends)
   - Определить локальный тренд (D1/H1) (см. @trends)
   - Оценить силу тренда (ценовое действие + объемы + ATR) (см. @trends)
   - Определить фазу рынка (см. @trends)
   - Проверить критерии
3. Отфильтровать инструменты, прошедшие все критерии
4. Вернуть список кандидатов (10-20 инструментов)
