# Точки входа и выхода

## Проверка перед входом

### 1. Анализ глобального и локального тренда

**Обязательно проверить:**
- **Глобальный тренд** (W1/MN): определить общее направление рынка
- **Локальный тренд** (D1/H1): определить краткосрочное направление
- **Синхронизация:** локальный тренд должен не противоречить глобальному
- **Приоритет:** предпочитать сделки в направлении глобального тренда

**Правило:**
- Если глобальный тренд восходящий, искать возможности для лонга
- Если локальный тренд противоречит глобальному, быть осторожным или пропустить сделку

### 2. Оценка силы тренда

**Проверить:**
- **Ценовое действие:** переписывание экстремумов, угол наклона, глубина откатов
- **Объемы:** контрастный объем (2-3x среднего) подтверждает силу
- **ATR:** потребление ATR, расширение/сжатие
- **Комплексная оценка:** сильный тренд = ценовое действие + высокие объемы + потребление ATR

### 3. Определение фазы рынка

**Проверить текущую фазу:**
- Накопление (Accumulation)
- Импульс (Impulse)
- Коррекция (Rollback)
- Ретест (Retest)
- Флэт/Рейндж (Flat/Range) — **75-80% времени рынок во флэте**
- Истощение (Exhaustion)

**Правило:**
- Избегать торговли в середине диапазона (флэт)
- Торговать только у границ диапазона или при четком пробое

## Триггеры входа

### 1. Пробой уровня

**Условия**:
- Цена пробила уровень поддержки/сопротивления
- **Объём выше среднего (подтверждение)** — контрастный объем (2-3x среднего) предпочтителен
- Закрытие свечи за уровнем
- **Проверка глобального тренда:** пробой должен быть в направлении глобального тренда

**Вход**: После пробоя, при ретесте уровня (или сразу, если сильный пробой с высоким объемом)

### 2. Отскок от уровня

**Условия**:
- Цена подошла к уровню поддержки/сопротивления
- Образовалась разворотная свеча (hammer, doji, engulfing)
- **Объём подтверждает разворот** — желателен контрастный объем
- **Проверка глобального тренда:** отскок должен быть в направлении глобального тренда

**Вход**: На закрытии разворотной свечи или на следующей свече

### 3. Пробой с ретестом

**Условие**: Цена пробила уровень, затем вернулась к нему для теста

**Вход**: При отскоке от уровня (ретest подтвердил уровень)

## Расчет стоп-лосса

### Использование ATR для расчета стоп-лоссов

**Расчет ATR:**
- **Период: 5 дней** (5 баров на дневном таймфрейме D1)
- Исключать паранормальные бары (>1.8-2x стандарта) и маленькие бары (<0.3-0.5x стандарта)
- Использовать расчетный ATR (без паранормальных баров) для статистики

**Расстояние от входа на основе ATR:**
- **Минимум:** 0.5-1.0 ATR от входа
- **Стандарт:** 1.0-1.5 ATR от входа
- **Широкий (для волатильных):** 1.5-2.0 ATR от входа

**Адаптация к волатильности:**
- Высокая волатильность (текущий ATR > среднего): шире стопы (1.5-2.0 ATR)
- Низкая волатильность (текущий ATR < среднего): уже стопы (0.5-1.0 ATR)

### Приоритет технического стопа

**ВАЖНОЕ ПРАВИЛО:**
- **Предпочитать технический стоп (за уровнем), если он не превышает расчетный (на основе ATR)**

**Логика выбора:**

1. **Рассчитать технический стоп:**
   - Для лонга: чуть ниже уровня поддержки
   - Для шорта: чуть выше уровня сопротивления

2. **Рассчитать расчетный стоп:**
   - На основе ATR: **20-30% от ATR** (например, если ATR = $1.00, то расчетный стоп = $0.20 - $0.30)
   - Учитывать волатильность инструмента

3. **Сравнить и выбрать:**
   - Если **технический стоп ≤ расчетный стоп** → использовать **технический стоп**
   - Если **технический стоп > расчетный стоп** → пересмотреть точку входа или использовать расчетный стоп (но не превышать максимальный риск 1%)

**Пример:**
- Вход в лонг: $185.60
- Уровень поддержки: $184.90
- Технический стоп: $184.80 (за уровнем)
- ATR(5): $0.70
- Расчетный стоп: 20-30% от ATR = $0.14 - $0.21, значит стоп на уровне $185.60 - $0.20 = $185.40 (используем 30% = $0.21, округляем до $0.20)
- **Выбор:** Технический стоп $184.80 < расчетный $185.40 → использовать технический $184.80

### Правила размещения SL

1. **За уровнем**: SL должен быть за уровнем поддержки/сопротивления
2. **Минимум 0.2%**: SL не должен быть меньше 0.2% от цены входа
3. **Минимум 0.5 ATR**: SL не должен быть ближе 0.5 ATR от входа (избежать рыночного шума)
4. **Максимум 1% риска**: Риск на сделку не должен превышать 1% депозита
5. **Учитывать волатильность**: В волатильных инструментах SL может быть шире (1.5-2.0 ATR)
6. **Приоритет технического стопа**: Использовать технический стоп, если он не превышает расчетный

### Формула расчета размера позиции

```
Размер позиции = (Риск в $) / (Цена входа - Стоп-лосс)
```

Пример:
- Депозит: $10,000
- Риск: 1% = $100
- Вход: $185.60
- SL: $184.90
- Размер: $100 / ($185.60 - $184.90) = $100 / $0.70 = 142 акции

## Расчет тейк-профита

### Минимальное соотношение R:R

**Обязательно**: Минимум 2:1 (риск:прибыль)

Пример:
- Вход: $185.60
- SL: $184.90 (риск $0.70)
- TP1: $187.00 (прибыль $1.40, R:R 2:1)
- TP2: $189.50 (прибыль $3.90, R:R 5.5:1)

### Множественные тейк-профиты

Рекомендуется использовать несколько TP:
- **TP1**: Консервативный (2:1 или 3:1) — закрыть часть позиции
- **TP2**: Умеренный (4:1 или 5:1) — закрыть ещё часть
- **TP3**: Агрессивный (7:1 или выше) — оставить небольшую часть

### Размещение TP

TP размещать:
- У следующего уровня сопротивления/поддержки
- У круглых чисел
- На основе ATR (например, 2x ATR от входа)

## Выход из позиции

### Плановый выход

1. **Достижение TP**: Закрыть позицию при достижении тейк-профита
2. **Достижение SL**: Закрыть позицию при достижении стоп-лосса
3. **Частичное закрытие**: Закрыть часть позиции на TP1, остальное на TP2

### Досрочный выход

Допустимо выходить раньше, если:
- Изменились условия (важные новости)
- Уровень пробит в обратную сторону
- Сигнал разворота (разворотная свеча)

**Важно**: Досрочный выход должен быть обоснован, не эмоциональным решением.

## Анализ объемов для подтверждения входа

**Правила подтверждения объемом:**

1. **Пробой уровня:**
   - Высокий объем (контрастный, 2-3x среднего) = Сильное подтверждение
   - Низкий объем = Слабость, возможен ложный пробой

2. **Отскок от уровня:**
   - Объем подтверждает разворот = Сильный сигнал
   - Отсутствие объема = Слабость сигнала

3. **Продолжение тренда:**
   - Цена вверх + объем вверх = Сильный тренд
   - Цена вниз + объем вверх = Сильный нисходящий тренд
   - Падение объема = Возможное окончание движения

4. **Дивергенция объемов:**
   - Цена делает новые максимумы, но объем снижается = Предупреждающий сигнал
   - Рассмотреть возможность выхода или не входить

## Важные правила

1. **Проверять глобальный и локальный тренд** — предпочитать сделки в направлении глобального тренда
2. **Оценивать силу тренда** — ценовое действие + объемы + ATR
3. **Определять фазу рынка** — избегать торговли в середине диапазона
4. **Всегда использовать SL** — без исключений
5. **Приоритет технического стопа** — использовать технический стоп, если он не превышает расчетный (ATR)
6. **Минимум 2:1 R:R** — иначе не торговать
7. **Подтверждение объемом** — движения без объема подозрительны
8. **Рассчитывать позицию заранее** — до входа
9. **Следовать плану** — не менять SL/TP во время сделки (кроме обоснованных случаев)
