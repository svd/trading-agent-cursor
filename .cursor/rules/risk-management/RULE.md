# Управление рисками

## Основное правило

**Максимальный риск на сделку: 1% от депозита**

Это означает, что если стоп-лосс сработает, потеря составит не более 1% от общего депозита.

## Расчет размера позиции

### Формула

```
Размер позиции = (Риск в $) / (Цена входа - Стоп-лосс)
```

Где:
- **Риск в $** = Депозит × 1% (или меньше)
- **Цена входа** = Цена, по которой планируется вход
- **Стоп-лосс** = Цена стоп-лосса

### Пример

- Депозит: $10,000
- Риск: 1% = $100
- Вход: $185.60
- SL: $184.90
- Размер: $100 / ($185.60 - $184.90) = 142 акции

### Для фьючерсов

```
Размер позиции = (Риск в $) / ((Цена входа - Стоп-лосс) × Размер контракта)
```

Пример (ES фьючерс):
- Депозит: $10,000
- Риск: 1% = $100
- Вход: 5,200
- SL: 5,180
- Размер контракта: $50 за пункт
- Размер: $100 / ((5,200 - 5,180) × $50) = $100 / $1,000 = 0.1 контракта (округлить до 1)

## Максимальный риск на день

**Рекомендация**: Не более 3-5% от депозита в день

Это означает:
- Если риск на сделку 1%, можно открыть максимум 3-5 сделок
- Если одна сделка убыточна, можно открыть ещё 2-4
- Если 2 сделки убыточны, можно открыть ещё 1-3

**Важно**: После 3 убыточных сделок подряд — прекратить торговлю на день.

## Управление капиталом

### Распределение капитала

1. **Торговый капитал**: 80-90% депозита
2. **Резерв**: 10-20% депозита (на случай форс-мажора)

### Размер позиции в зависимости от уверенности

- **Высокая уверенность** (все предпосылки выполнены): 1% риска
- **Средняя уверенность**: 0.5-0.7% риска
- **Низкая уверенность**: Не торговать или 0.3% риска

### Учет волатильности (ATR) при расчете размера позиции

**Важно:**
- При расчете размера позиции учитывать ATR инструмента
- Если ATR высокий, стоп будет шире, размер позиции должен быть меньше (при фиксированном риске 1%)
- Если ATR низкий, стоп может быть уже, размер позиции может быть больше (при фиксированном риске 1%)

**Формула остается той же:**
```
Размер позиции = (Риск в $) / (Цена входа - Стоп-лосс)
```

Но стоп-лосс рассчитывается с учетом ATR и приоритета технического стопа.

## Правила стоп-лосса

### Использование ATR для размещения стоп-лоссов

**Расчет ATR:**
- **Период: 5 дней** (5 баров на дневном таймфрейме D1)
- Исключать паранормальные бары (>1.8-2x стандарта) и маленькие бары (<0.3-0.5x стандарта)

**Расстояние от входа:**
- **Минимум:** 0.5-1.0 ATR от входа
- **Стандарт:** 1.0-1.5 ATR от входа
- **Широкий (для волатильных):** 1.5-2.0 ATR от входа

**Адаптация к волатильности:**
- Высокая волатильность (текущий ATR > среднего): шире стопы (1.5-2.0 ATR)
- Низкая волатильность (текущий ATR < среднего): уже стопы (0.5-1.0 ATR)

**Учет фазы рынка:**
- В импульсе: более широкие стопы (1.5-2.0 ATR)
- В накоплении: стандартные стопы (1.0-1.5 ATR)
- Во флэте: избегать торговли или использовать узкие стопы

### Приоритет технического стопа

**ВАЖНОЕ ПРАВИЛО:**
- **Предпочитать технический стоп (за уровнем), если он не превышает расчетный (на основе ATR)**

**Логика выбора:**

1. **Рассчитать технический стоп:**
   - Для лонга: чуть ниже уровня поддержки
   - Для шорта: чуть выше уровня сопротивления

2. **Рассчитать расчетный стоп:**
   - На основе ATR: **20-30% от ATR** (например, если ATR = $1.00, то расчетный стоп = $0.20 - $0.30)
   - Учитывать волатильность инструмента

3. **Сравнить и выбрать:**
   - Если **технический стоп ≤ расчетный стоп** → использовать **технический стоп**
   - Если **технический стоп > расчетный стоп** → пересмотреть точку входа или использовать расчетный стоп (но не превышать максимальный риск 1%)

**Пример:**
- Вход в лонг: $185.60
- Уровень поддержки: $184.90
- Технический стоп: $184.80 (за уровнем)
- ATR(5): $0.70
- Расчетный стоп: 20-30% от ATR = $0.14 - $0.21, значит стоп на уровне $185.60 - $0.20 = $185.40 (используем 30% = $0.21, округляем до $0.20)
- **Выбор:** Технический стоп $184.80 < расчетный $185.40 → использовать технический $184.80

### Основные правила

1. **Всегда использовать SL** — без исключений
2. **SL за уровнем** — минимум 0.2% от цены, минимум 0.5 ATR от входа
3. **Приоритет технического стопа** — использовать технический стоп, если он не превышает расчетный (ATR)
4. **Не двигать SL против позиции** — только в сторону прибыли (trailing stop)
5. **Не увеличивать риск** — если SL слишком широкий, уменьшить размер позиции
6. **Учитывать ATR** — адаптировать стопы к волатильности инструмента

## Правила тейк-профита

1. **Минимум 2:1 R:R** — иначе не торговать
2. **Частичное закрытие** — закрыть часть на TP1, остальное на TP2
3. **Trailing stop** — после достижения TP1, двигать SL в безубыток

## Эмоциональный контроль

### Правила для предотвращения эмоциональных решений

1. **План до сессии** — все сценарии расписаны заранее
2. **Следовать плану** — не менять SL/TP во время сделки
3. **Лимит сделок** — максимум 5 сделок в день
4. **Перерыв после убытка** — минимум 30 минут после стоп-лосса

### Признаки эмоциональной торговли

- Увеличение размера позиции после убытка (revenge trading)
- Удаление стоп-лосса
- Торговля без плана
- Торговля больше запланированного количества сделок

**Если заметил эти признаки — прекратить торговлю на день.**

## Важные правила

1. **1% риска на сделку** — максимум
2. **3-5% риска в день** — максимум
3. **Всегда использовать SL** — без исключений
4. **Минимум 2:1 R:R** — иначе не торговать
5. **Следовать плану** — не менять во время сделки
