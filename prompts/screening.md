# Скрининг инструментов

## Задача

Найти инструменты, подходящие для торговли сегодня по методологии Герчика.

## Критерии отбора (из базы знаний)

**Для акций (US Market)**:
1. **Объём**: Average Volume > 300,000 (стандарт) или > 500,000 (высокая ликвидность). Избегать инструментов с объёмом < 300,000 (или < 1M для penny stocks)
2. **Цена**: Фильтр для акций >$0.50 (чтобы 100 акций > $50). Отдельные списки для акций >$10 и <$10
3. **Рынок**: Только США (исключить ADR), исключить ETF (кроме SPY, IWM)
4. **Волатильность**: Average True Range > 1 доллар (Over 1)
5. **Правило отклонения**: Строго применять правило "95%" — **95%** просмотренных графиков должны быть отклонены. Торговать только 5% с "супер понятной картинкой"
6. **Фильтрация**: Выбирать только "жесткие" ситуации. Исключать "зараженные зоны", глубокие ложные пробои, "пилу"

**Для криптовалют**:
1. **Ликвидность**: Фильтр для ликвидности > $1,000,000 (или >100 BTC для пар)
2. **Пороги объёма**:
   - Риск $100: Объём **> 60–70 миллионов**
   - Риск $200+: Объём **> 100 миллионов**
   - Минимум: **> 50 миллионов** (манипулятивные инструменты)
3. **Проверка спота**: Для монет с объёмом 50M на фьючерсах, объём спота должен быть **15–20 миллионов+**

**Для Forex**:
1. Начать с 10–15 пар. Избегать торговли одной и той же парой каждый день; ротировать для поиска активных рынков
2. **Формула ликвидности**: `Коэффициент = (0.1 * ATR_Daily) / Spread`. Порог должен быть **> 4** (идеально **> 5**)

**Общие критерии** (см. @trends):
1. **Глобальный тренд (W1/MN)**: Определить общее направление рынка на старших таймфреймах (см. @trends)
2. **Локальный тренд (D1/H1)**: Чёткий тренд на D1 или H4, либо инструмент у границы диапазона (см. @trends)
3. **Сила тренда**: Оценить силу тренда (ценовое действие + объемы + ATR) (см. @trends)
4. **Фаза рынка**: Определить текущую фазу (накопление/импульс/коррекция/ретест/флэт/истощение) — избегать инструментов во флэте в середине диапазона (см. @trends)

## Процесс

1. Получить список инструментов для скрининга (или использовать предопределённый список)
2. Для каждого инструмента:
   - Получить текущую котировку (`get_real_time_quote`) — проверить цену, спред
   - Получить исторические данные (`get_stock_history` для анализа тренда на разных таймфреймах: W1, D1, H1)
   - Получить статистику (`get_market_stats` для ATR, Average Volume, текущего объёма)
   - **Проверить базовые критерии**:
     - Для акций: Average Volume > 300k (стандарт) или > 500k (высокая ликвидность)
     - Для акций: ATR > $1
     - Для акций: Цена >$0.50 (или >$10 для основного списка)
     - Для акций: Рынок — только США (исключить ADR, ETF кроме SPY, IWM)
     - Для криптовалют: Ликвидность > $1M, объём согласно порогам
     - Для Forex: Рассчитать коэффициент ликвидности = (0.1 * ATR_Daily) / Spread, должен быть > 4
   - Определить глобальный тренд (W1/MN) (см. @trends)
   - Определить локальный тренд (D1/H1) (см. @trends)
   - Оценить силу тренда (ценовое действие + объемы + ATR) (см. @trends)
   - Определить фазу рынка (см. @trends)
   - **Применить правило 95%**: Отклонить инструмент, если картинка не "супер понятная"
   - **Фильтрация**: Исключить "зараженные зоны", глубокие ложные пробои, "пилу"
3. **Проверка корреляции**:
   - Для акций: проверить корреляцию с основным индексом (S&P 500 для акций США, NASDAQ для технологических акций)
   - Предпочтение инструментам, которые **не коррелируют** с индексом и имеют "собственную жизнь"
   - Проверить синхронность индексов (Dow Jones, Nasdaq, S&P 500) — дивергенция между индексами = сигнал риска
   - Для некоторых инструментов (например, Apple) корреляция с индексом критична; для InPlay (летящих на новостях) корреляция вторична
   - Если рынок падает (S&P 500 в минусе), избегать лонгов в слабых акциях
   - Использовать `get_stock_history` для сравнения динамики инструмента с индексом
4. Отфильтровать инструменты, прошедшие все критерии
5. Вернуть список кандидатов (10-20 инструментов)

## Формат вывода

```markdown
# Скрининг инструментов - {DATE}

## Критерии
- **Акции**: Average Volume > 300k (стандарт) / > 500k (высокая ликвидность), ATR > $1, цена >$0.50, только США
- **Криптовалюты**: Ликвидность > $1M, объём > 50-100M (в зависимости от риска)
- **Forex**: Коэффициент ликвидности > 4 (идеально > 5)
- **Правило 95%**: Торговать только 5% с "супер понятной картинкой"
- Глобальный тренд (W1/MN): Восходящий/нисходящий/боковой (см. @trends)
- Локальный тренд (D1/H1): Чёткий тренд или граница диапазона (см. @trends)
- Сила тренда: Сильный/средний/слабый (см. @trends)
- Фаза рынка: Накопление/импульс/коррекция/ретест/флэт/истощение (см. @trends)
- Корреляция: Инструмент имеет "собственную жизнь" или низкая корреляция с индексом (для акций)
```
